# 公众号爆款文章创作方法论

**AI-Powered WeChat Article Creation Methodology**

基于真实创作案例《AI 扔下的生肉，谁来煮？》提炼的完整方法论。

## 核心公式

```
爆款 = 多篇融合（3-5篇） 
     + 人设注入（真实案例+数据） 
     + 坚决利他（70%方法论） 
     + 文笔润色（画面感+文学感） 
     + 对抗式审核（≥8分才发布）
```

## 五大核心要素

### 1. 多篇爆款融合
从 3-5 篇相关文章中提取核心观点，重新组合。不是复制文字，而是提取观点用自己的话重新表达。

### 2. 注入个人人设信息
让文章成为"只有你能写出来"的作品。用真实案例替代抽象观点，用具体数字替代模糊描述。

### 3. 坚决利他
70% 讲"怎么办"，30% 讲"是什么"。每个方法都必须有"具体操作"小节。

### 4. 文笔润色
- 用画面替代抽象
- 用对话替代叙述
- 用细节替代形容词
- 用数字替代模糊描述
- 变化句子长度

### 5. 对抗式内容审核
用毒舌参谋的视角，5 维度攻击框架：
1. 标题吸引力（X/10）
2. 结构逻辑（X/10）
3. 数据支撑（X/10）
4. 读者视角（X/10）
5. 篇幅比例（X/10）

**评分标准：**
- 9-10 分：✅ 直接通过
- 8 分：✅ 通过（可带轻微修改建议）
- 6-7 分：🔁 打回，需要修改
- ≤5 分：🔁 重写

## 完整创作流程（7 步）

1. **素材收集**（30 分钟）：收集 3-5 篇相关文章，提取核心信息
2. **人设档案**（15 分钟）：整理个人背景、经历、数据
3. **初稿创作**（60 分钟）：基于素材和人设创作
4. **对抗式审核**（30 分钟）：5 维度攻击框架严格审核
5. **完全重写**（90 分钟）：解决致命问题
6. **二次审核**（20 分钟）：确认改进，期望 ≥8/10 分
7. **最终检查**（15 分钟）：用检查清单验证

## 常见陷阱

### 陷阱 1：观点都是二手的
**症状**：全文都在转述别人的故事，没有作者自己的一手经验。

**解决方案**：列出作者的 5 个真实经历，每个经历要有时间、地点、具体数据、情感波动。

### 陷阱 2：讲"是什么"太多，讲"怎么办"太少
**症状**：70% 在讲"AI 很快，世界在变"，只有 5% 在讲"你该怎么做"。

**解决方案**：倒过来，30% 讲现象，70% 讲方法。每个方法都要有"具体操作"小节。

### 陷阱 3：情感不真实
**症状**：凌晨三点的场景太文学化，情绪描写像在表演。

**解决方案**：用具体细节替代抽象描写，用对话替代叙述，用数字替代形容词。

### 陷阱 4：广告位太刻意
**症状**："我在做醒觉社的时候"出现 3 次，每次提到产品都很生硬。

**解决方案**：产品名称最多出现 2 次，删掉所有"我在做 XX 的时候"句式。

### 陷阱 5：深度思考文的特殊陷阱
当用户要求"深度思考文"而非"工具指南"时：
- 案例真实性危机（10 个"朋友告诉我"但无可验证信息）
- 作者实践经验不足（只有观察没有实践）
- 立场摇摆（既说好又说坏）
- 结尾逃避（"我不知道答案"+"你自己想"）
- 想说的太多，每个都没说透

## 成功案例

### 《AI 扔下的生肉，谁来煮？》

**创作数据**：
- 素材：3 篇文章
- 初稿：4200 字，耗时 2 小时
- 审核：6/10 分，打回重写
- 重写：3400 字，耗时 1.5 小时
- 二次审核：预计 8-9/10 分

**关键改进**：

| 维度 | 第一版 | 重写版 |
|------|--------|--------|
| 字数 | 4,200 | 3,400 |
| Andy 真实案例 | 0 个 | 5+ 个 |
| 具体数据 | 2 处 | 10+ 处 |
| "怎么办"占比 | 5% | 70% |
| 广告位 | 3 处 | 0 处 |
| 审核分数 | 6/10 | 8-9/10 |

### 《人类正在进化成三体人》

**迭代过程**：5 次迭代，从 4200 字 4.5 分 → 2650 字 8.5+ 分

**关键经验**：
1. 对抗式审核是核心（每次迭代都必须有 5 维度审核）
2. 方法论必须前置（在 1500 字处出现）
3. 哲学追问不能超过 15%（超过会成为弃读高峰区）
4. 篇幅控制黄金比例（理想字数 2500-3000 字）
5. 品牌露出最多 2 次（超过会显得像广告）

## 核心原则

**记住**：
- 真诚 > 文采
- 方法 > 观点
- 删减 > 添加
- 审核 > 直觉

**最重要的一条**：
如果你自己都不想分享到朋友圈，那就还没写好。
